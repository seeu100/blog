  • Geneious 9.0.2 for Mac OS 10.7+

  • Geneious 9.0.2 for Linux 64-bit

chmod +x Geneious_linux64_9_0_2_with_jre.sh
./Geneious_linux64_9_0_2_with_jre.sh
